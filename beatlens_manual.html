1. Pendahuluan

Apa itu BeatLens?

BeatLens adalah aplikasi rekomendasi musik cerdas yang menggunakan teknologi Hybrid AI (Rule-based + K-Nearest Neighbors Machine Learning) untuk memberikan rekomendasi lagu yang sesuai dengan mood dan preferensi musik Anda.

Fitur Utama

  • üé≠ 5 Kategori Mood: Sedih, Happy, Galau, Chill, Semangat
  • üé∏ 10+ Genre Musik: Pop, Rock, Indie, Ballad, EDM, dan lainnya
  • ü§ñ AI-Powered: Algoritma KNN untuk rekomendasi yang akurat
  • ‚ö° Cepat: Response time < 10ms
  • üìä 154+ Lagu: Database yang terus berkembang

Teknologi yang Digunakan

  • Backend: Python, FastAPI, SQLite
  • Frontend: Next.js, React, Tailwind CSS
  • AI/ML: Scikit-learn, KNN Algorithm
  • Integration: Spotify Web API

2. Instalasi

Persyaratan Sistem

Komponen Requirement
Python 3.8 atau lebih tinggi
Node.js 16 atau lebih tinggi
npm 8 atau lebih tinggi
RAM Minimal 2GB
Storage Minimal 500MB

Langkah Instalasi

A. Clone Repository

git clone https://github.com/ypratama123/BeatLens.git
cd BeatLens

B. Setup Backend

1. Masuk ke folder backend

cd backend

2. Install dependencies

pip install -r requirements.txt

3. Inisialisasi database

python db/init_db.py

4. Jalankan server backend

python app.py
Backend akan berjalan di: http://localhost:8000

C. Setup Frontend

1. Buka terminal baru, masuk ke folder frontend

cd frontend

2. Install dependencies

npm install

3. Jalankan development server

npm run dev
Frontend akan berjalan di: http://localhost:3000

D. Verifikasi Instalasi

  1. Cek Backend: Buka http://localhost:8000/health
    Harus menampilkan: {"status": "healthy", "database": "connected"}
  2. Cek Frontend: Buka http://localhost:3000
    Harus menampilkan halaman landing BeatLens

4. Fitur-Fitur

4.1 Sistem Rekomendasi Hybrid AI

Rule-Based Engine

  • Memfilter lagu berdasarkan aturan mood
  • Setiap mood memiliki preferensi genre dan tempo
  • Contoh: Mood "Sedih" ‚Üí Genre: Indie, Ballad | Tempo: Slow, Medium

K-Nearest Neighbors (KNN)

  • Menghitung similarity antara preferensi user dan lagu
  • Menggunakan encoding untuk genre, mood, dan tempo
  • Mengurutkan hasil berdasarkan similarity score

4.2 Database Musik

Statistik Database

  • Total Lagu: 154+
  • Genre: 10 kategori
  • Mood: 5 kategori
  • Distribusi Seimbang: Setiap mood memiliki 25+ lagu

Contoh Lagu Populer

  • Sedih: Someone Like You (Adele), Fix You (Coldplay)
  • Happy: Blinding Lights (The Weeknd), Happy (Pharrell)
  • Galau: Someone You Loved (Lewis Capaldi), Heather (Conan Gray)
  • Chill: Weightless (Marconi Union), Holocene (Bon Iver)
  • Semangat: Eye of the Tiger (Survivor), Believer (Imagine Dragons)

4.4 Integrasi Spotify

Setup Spotify API (Opsional)

5. Troubleshooting

Masalah Umum dan Solusi

Backend Tidak Bisa Dijalankan

Error: "ModuleNotFoundError"

# Solusi: Install ulang dependencies
pip install -r requirements.txt

Error: "Database not found"

# Solusi: Inisialisasi database
python db/init_db.py

Error: "Port 8000 already in use"

# Solusi: Matikan proses yang menggunakan port 8000
# Windows:
netstat -ano | findstr :8000
taskkill /PID <PID> /F

# Mac/Linux:
lsof -ti:8000 | xargs kill -9

Frontend Tidak Bisa Dijalankan

Error: "npm command not found"

Solusi: Install Node.js dari nodejs.org

Error: "Module not found"

# Solusi: Install ulang dependencies
rm -rf node_modules
npm install

Error: "Port 3000 already in use"

# Solusi: Gunakan port lain
npm run dev -- -p 3001

API Connection Failed

Error: "Failed to fetch" di browser console

  • Pastikan backend berjalan di http://localhost:8000
  • Cek CORS settings di backend/app.py
  • Test endpoint: http://localhost:8000/health

Tidak Ada Rekomendasi

Pesan: "Maaf, belum ada lagu yang cocok"

  • Coba kombinasi mood/genre yang berbeda
  • Hapus filter genre (biarkan kosong)
  • Cek database: python db/init_db.py

6. FAQ (Frequently Asked Questions)

Q1: Apakah BeatLens gratis?

A: Ya, BeatLens sepenuhnya gratis dan open source.

Q2: Apakah perlu koneksi internet?

A: Untuk menjalankan lokal, tidak perlu internet. Tapi untuk preview Spotify, perlu koneksi internet.

Q3: Bagaimana cara menambah lagu baru?

A: Edit file backend/db/seed_data.json, tambahkan lagu baru dengan format:

{
  "title": "Judul Lagu",
  "artist": "Nama Artis",
  "genre": "pop",
  "mood": "happy",
  "tempo": "fast",
  "spotify_id": "spotify_track_id"
}

Lalu jalankan: python db/init_db.py

Q4: Apakah bisa deploy ke production?

A: Ya! Lihat file DEPLOYMENT_GUIDE.md untuk panduan deployment ke Vercel, Railway, atau Render.

Q5: Bagaimana cara mendapat Spotify ID?

A:

  1. Buka lagu di Spotify Web Player
  2. Klik "Share" ‚Üí "Copy Song Link"
  3. URL format: https://open.spotify.com/track/SPOTIFY_ID
  4. Ambil bagian SPOTIFY_ID

Q6: Apakah data saya disimpan?

A: Tidak. BeatLens tidak menyimpan data personal atau riwayat pencarian Anda.

Q7: Kenapa rekomendasi tidak sesuai?

A: Sistem AI terus belajar. Coba kombinasi mood/genre berbeda untuk hasil yang lebih sesuai.

Q8: Apakah bisa digunakan offline?

A: Ya, untuk fitur rekomendasi. Tapi preview Spotify memerlukan internet.

Q9: Bagaimana cara update aplikasi?

A:

git pull origin main
cd backend && pip install -r requirements.txt
cd ../frontend && npm install

Q10: Dimana saya bisa melaporkan bug?

A: Buat issue di GitHub: https://github.com/ypratama123/BeatLens/issues
